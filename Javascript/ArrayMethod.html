Array Method

tostring : Convert array into string POP: Remove the last element from the array list

PUSH : Add a new element in the last of the array

Shift : Remove the first element from beginning of the array

Unshift : Add an element from the beginning of the array

Concat : Merging existing array into the array

Sort : sort an array in ascending order

Reverse : Reverse an array

Map Filter Reduce: map method used to creates a new array from an existing one. perform a function on each element
